# uni-app 电商模板

自己研究开发的纯uni-app电商前端模板，会不定期更新模板内容，功能，UI界面的。
#### 如果对您有用 好评哦 (^ω^)

## 说明
1. css部分使用了uni.scss部分变量, 其他语法已剔除特殊语法, 除变量外可直接替换其他css预处理器
2. app.vue下定义了个别全局样式以及引用iconfont图标
3. 该项目为前端页面效果展示, 数据和图片来自网络，均只用于案例展示，可以删除，文件在/static/img，请勿用于其他用途。
